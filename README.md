# ninja-toolkit

## Tools for NinjaMountain DevOps teams

### Included:

- Ubuntu Server 20.04 LTS
- Docker
- Helm
- Kubectl
- Hashicorp Packer
- Hashicorp Terraform
- Hashicorp Vault
- Hashicorp Consul
- Ansible
- Google Cloud SDK

### Usage:

Create
```
vagrant up
```
Access
```
vagrant ssh
```
Reprovision
```
vagrant provision
```
Destroy
```
vagrant destroy
```
Creation
```
vagrant up
```
